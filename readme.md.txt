## Descricao

Trabalho desenvolvido para a matéria de Sistemas Distribuídos.
Sistema de Transação de Moedas em ambiente distribuído. Com banco de dados distribuído, mensagens em broadcast e unicast, eleição de minerador e segurança assimétrica.
 
## Code Example

Rodar no Eclipse 4x a Classe main para que os processos filhos sejam executados.
